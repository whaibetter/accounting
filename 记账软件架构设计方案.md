# 记账软件架构设计方案（轻量版）

> 定位：个人使用的轻量记账工具，Python后端，核心功能优先，极简部署。

## 1. 系统架构设计

### 1.1 整体架构

前后端分离，三端共用一套Python API。个人使用无需复杂的安全和优化机制。

```
┌───────────────────────────────────────────────┐
│                   客户端层                     │
│   ┌──────────┐ ┌──────────┐ ┌──────────────┐ │
│   │  网页端   │ │ 安卓端   │ │  微信小程序   │ │
│   │ (Vue3)   │ │(Kotlin)  │ │  (原生)      │ │
│   └────┬─────┘ └────┬─────┘ └──────┬───────┘ │
│        └─────────────┼──────────────┘         │
│                      │ HTTP                    │
└──────────────────────┼────────────────────────┘
                       │
              ┌────────▼────────┐
              │   FastAPI 应用   │
              │   (Python 3.11) │
              │                 │
              │  ┌───────────┐  │
              │  │  Router   │  │
              │  ├───────────┤  │
              │  │  Service  │  │
              │  ├───────────┤  │
              │  │  CRUD     │  │
              │  └───────────┘  │
              └────────┬────────┘
                       │
              ┌────────▼────────┐
              │    SQLite 数据库 │
              │   (单文件存储)   │
              └─────────────────┘
```

### 1.2 技术栈选型

| 层级 | 技术 | 选型理由 |
|------|------|----------|
| **网页端** | Vue 3 + TypeScript + Vant 4 | 组件化开发，Vant适配移动端 |
| **安卓端** | Kotlin + Jetpack Compose | 现代Android开发方式 |
| **小程序** | 微信原生小程序 + TypeScript | 性能最优 |
| **后端框架** | FastAPI (Python 3.11) | 异步高性能，自动生成API文档，开发效率极高 |
| **ORM** | SQLAlchemy 2.0 | Python生态最成熟的ORM，支持异步 |
| **数据验证** | Pydantic V2 | 与FastAPI深度集成，自动校验请求参数 |
| **数据库** | SQLite | 零配置，单文件，个人使用完全够用，无需安装数据库服务 |
| **部署** | Uvicorn | ASGI服务器，直接运行即可 |

## 2. 数据库设计

### 2.1 ER关系图

```
┌──────────┐     1:N     ┌──────────┐     1:N     ┌──────────┐
│  Account │─────────────│   Bill   │─────────────│ BillTag  │
│ (资金账户)│             │  (账单)   │             │(账单-标签)│
└──────────┘             └──────────┘             └──────────┘
                               │                        │
                               │ N:1                    │ N:1
                               ▼                        ▼
                         ┌──────────┐            ┌──────────┐
                         │ Category │            │   Tag    │
                         │  (分类)   │            │  (标签)   │
                         └──────────┘            └──────────┘
                               │
                               │ 自关联(parent_id)
                               ▼
                         ┌──────────┐
                         │ Category │
                         │ (子分类)  │
                         └──────────┘
```

> 个人使用无需用户表，所有数据直接归属单用户。

### 2.2 表结构设计

#### 2.2.1 资金账户表 (account)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 账户ID |
| name | VARCHAR(50) | NOT NULL | 账户名称 |
| type | INTEGER | NOT NULL, DEFAULT 1 | 类型: 1-现金, 2-银行卡, 3-信用卡, 4-支付宝, 5-微信, 6-其他 |
| icon | VARCHAR(50) | | 图标标识 |
| color | VARCHAR(20) | | 颜色标识 |
| balance | REAL | NOT NULL, DEFAULT 0 | 当前余额 |
| initial_balance | REAL | NOT NULL, DEFAULT 0 | 初始余额 |
| sort_order | INTEGER | NOT NULL, DEFAULT 0 | 排序序号 |
| is_default | INTEGER | NOT NULL, DEFAULT 0 | 是否默认账户 |
| status | INTEGER | NOT NULL, DEFAULT 1 | 状态: 1-正常, 0-已归档 |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

#### 2.2.2 分类表 (category)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 分类ID |
| parent_id | INTEGER | FK, DEFAULT NULL | 父分类ID(NULL为顶级分类) |
| name | VARCHAR(50) | NOT NULL | 分类名称 |
| type | INTEGER | NOT NULL | 类型: 1-支出, 2-收入 |
| icon | VARCHAR(50) | | 图标标识 |
| sort_order | INTEGER | NOT NULL, DEFAULT 0 | 排序序号 |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

#### 2.2.3 标签表 (tag)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 标签ID |
| name | VARCHAR(50) | NOT NULL, UNIQUE | 标签名称 |
| color | VARCHAR(20) | | 颜色标识 |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |

#### 2.2.4 账单表 (bill)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 账单ID |
| account_id | INTEGER | FK, NOT NULL | 资金账户ID |
| category_id | INTEGER | FK, NOT NULL | 分类ID |
| type | INTEGER | NOT NULL | 类型: 1-支出, 2-收入, 3-转账 |
| amount | REAL | NOT NULL | 金额(绝对值) |
| bill_date | DATE | NOT NULL | 账单日期 |
| bill_time | TIME | | 账单时间(可空) |
| remark | VARCHAR(500) | | 备注 |
| transfer_to_account_id | INTEGER | FK | 转入账户ID(转账类型时) |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引**:
- idx_bill_date (bill_date)
- idx_bill_category (category_id)
- idx_bill_account (account_id)
- idx_bill_type (type)

#### 2.2.5 账单-标签关联表 (bill_tag)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | ID |
| bill_id | INTEGER | FK, NOT NULL | 账单ID |
| tag_id | INTEGER | FK, NOT NULL | 标签ID |

**唯一约束**: (bill_id, tag_id)

```
┌──────────────────────────────────────┐
│          记账软件功能模块(轻量版)       │
├──────────┬──────────┬────────────────┤
```

### 3.2 各模块职责

#### 3.2.1 账户模块 (account)
- 资金账户CRUD
- 账户余额调整
- 默认账户设置
- 账户归档

#### 3.2.2 账单模块 (bill)
- 账单CRUD（快速记账）
- 转账记录（账户间转账）

- 导出为Excel格式（.xlsx）

## 4. API接口规范

### 4.1 通用规范

**基础URL**: `http://localhost:8000/api/v1`

**统一响应格式**:
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

**错误码**:

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 404 | 资源不存在 |
| 422 | 业务逻辑错误 |
| 500 | 服务器内部错误 |

**分页参数**: `?page=1&size=20`

**分页响应**:
```json
{
  "code": 200,
  "data": {
    "items": [],
    "total": 100,
    "page": 1,
    "size": 20
  }
}
```

## 5. 部署方案

### 5.1 本地运行（开发/个人使用）

```bash
# 安装依赖
cd server
pip install -r requirements.txt

# 启动服务
python run.py

# 访问
# API: http://localhost:8000
# 自动文档: http://localhost:8000/docs
```

**run.py**:
```python
import uvicorn

if __name__ == "__main__":
    uvicorn.run("app.main:app", host="0.0.0.0", port=8000, reload=True)
```

**requirements.txt**:
```
fastapi==0.110.0
uvicorn==0.29.0
sqlalchemy==2.0.29
pydantic==2.6.4
openpyxl==3.1.2
```

### 5.2 局域网访问（手机/平板连接）

```bash
# 启动时绑定0.0.0.0，同一局域网设备可通过IP访问
python run.py
# 其他设备访问: http://192.168.x.x:8000
```

### 5.3 数据备份

SQLite数据库为单文件，备份只需复制文件：

```bash
# 手动备份
cp data/accounting.db data/accounting_backup_20260417.db

# 或使用脚本定时备份
```

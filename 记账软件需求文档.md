# 记账软件需求文档

## 1. 产品概览

本产品是一款面向普通个人用户的记账软件，旨在解决用户记账繁琐、数据分散、缺乏分析等核心痛点。通过提供简洁直观的用户界面和多端同步功能，帮助用户轻松管理个人财务，获得清晰的财务洞察。

产品支持网页端、安卓端和小程序三个平台，共用一套后端系统，确保数据的一致性和实时性。

## 2. 核心功能

### 2.1 基础记账
- **快速记账**：支持快速录入收支记录，减少操作步骤
- **分类管理**：支持自定义收支分类，方便统计分析
- **标签管理**：支持为记账条目添加标签，便于多维度分析
- **备注功能**：支持为记账条目添加详细备注
- **批量操作**：支持批量导入、导出或修改记账数据
- **周期性记账**：支持设置固定周期的收支，如房租、水电费等
- **项目记账**：支持为特定项目单独记账，如旅行、装修等

### 2.2 多端同步
- **实时同步**：数据在不同端之间实时更新，确保数据一致性
- **离线使用**：支持离线记账，网络恢复后自动同步
- **数据备份**：支持手动或自动备份数据，防止数据丢失
- **数据迁移**：支持从其他记账软件导入数据

### 2.3 数据分析
- **收支概览**：总收支、结余等基本财务状况
- **分类统计**：按分类统计收支情况，如餐饮、交通等
- **趋势分析**：收支趋势变化，如月度、年度对比
- **预算执行**：预算使用情况和执行率
- **余额趋势**：实时追踪账户余额变化，预测未来财务状况

### 2.4 AI智能记账
- **AI智能分类**：通过AI大模型自动识别账单分类，智能分析消费习惯
- **自然语言记账**：支持用自然语言描述支出，AI自动解析并创建账单
- **LLM配置**：支持配置不同的AI大模型（OpenRouter兼容API）

### 2.5 账户管理
- **多账户支持**：支持管理多个账户（现金、银行卡、信用卡等）
- **账户余额**：自动计算各账户余额，支持账户转账

### 2.6 预算管理
- **月度预算**：设置月度总预算和分类预算
- **预算提醒**：当支出接近预算时发送提醒
- **预算调整**：支持在预算执行过程中调整预算
- **预算分析**：分析预算执行情况，提供优化建议

### 2.7 安全与隐私
- **权限控制**：严格的权限管理，防止未授权访问
- **数据加密**：敏感数据加密存储
- **密码保护**：支持设置访问密码

## 3. 技术架构

### 3.1 系统架构
- **前端**：网页端采用 Vue 3 + TypeScript + Vite，安卓端采用 Kotlin + Jetpack Compose
- **后端**：自建服务器部署，采用 RESTful API 架构
- **数据库**：使用 SQLite 关系型数据库存储用户数据和记账信息
- **AI服务**：支持 OpenRouter 兼容的 LLM API

### 3.2 技术选型
| 端 | 技术栈 |
|---|--------|
| Web前端 | Vue 3 + TypeScript + Vite + Pinia + Vue Router |
| Android | Kotlin + Jetpack Compose + Material3 + ViewModel + Retrofit |
| 后端 | Python + FastAPI + SQLAlchemy + Pydantic |
| 数据库 | SQLite |
| AI | OpenRouter API (兼容多种大模型) |
| 部署 | Docker |

## 4. 用户界面设计

### 4.1 设计原则
- **简洁直观**：界面简洁明了，操作流程简单
- **响应式设计**：在不同设备上都能获得良好的使用体验
- **深色主题**：默认深色主题，减少视觉疲劳

### 4.2 页面结构

#### Web端
- **首页**：收支概览、快速记账入口、最近交易记录
- **账单页**：账单列表、筛选、编辑
- **记账页**：收支类型选择、金额输入、分类选择、标签添加、备注输入
- **统计页**：各种数据分析报表和图表
- **余额趋势页**：账户余额变化趋势图
- **账户页**：账户列表、余额统计
- **导入页**：批量导入数据
- **AI记账页**：AI智能识别、自然语言记账
- **LLM设置页**：配置AI大模型

#### Android端
- **首页**：收支概览、最近账单、快速记账
- **账单页**：账单列表
- **记账页**：新建账单
- **统计页**：收支统计
- **账户页**：账户管理

### 4.3 设计规范
- **主色调**：紫色系 (#818CF8)
- **背景色**：深色 (#0F0F13, #1E1E30)
- **收入色**：绿色 (#4CAF50)
- **支出色**：红色 (#F44336)
- **圆角**：卡片圆角 16dp，按钮圆角 10dp
